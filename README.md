# Content Creator - Event Content Service

A modern event content service where trained content creators attend events and capture high-quality photos & videos using iPhones, delivering fast, social-ready content.

**Positioning:** *"Professional event content, minus the stress."*

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Modern Next.js 14 with App Router
- Tailwind CSS for styling with custom color accents
- Video background support in Hero section
- Responsive design
- Mobile-friendly navigation
- Conversion-focused booking form
- Complete event content service website

## Website Sections

1. **Hero** - Main headline with clear value proposition
2. **How It Works** - 4-step process explanation
3. **Why iPhone** - Trust-building section with comparison table
4. **Event Types** - Visual grid of supported events
5. **Packages & Pricing** - Three-tier pricing structure
6. **Add-Ons** - Optional upgrades (on-the-night, post-event, premium)
7. **Testimonials** - Client reviews
8. **FAQ** - Event-specific questions
9. **Booking Form** - Conversion-focused booking system
